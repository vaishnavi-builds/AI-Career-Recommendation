# 🎯 AI Career Recommendation System

An AI-driven web application that analyzes user **skills, interests, education, and work preferences** to recommend the most suitable career paths along with learning roadmap, skills required, and future scope.


## 🚀 Features

* 🤖 AI-Driven Career Recommendations
* 🧠 Skill & Interest Based Matching
* 🎓 Education Level Selection
* 💼 Work Style Preferences (Remote, Hybrid, On-site)
* 📊 Match Score for Each Career
* 🛣️ Career Roadmap Generation
* 📚 Recommended Courses
* 🧩 Skills Gap Analysis
* 🔮 Future Scope of Career
* 💎 Professional Modern UI
* 📱 Fully Responsive Design


## 🖥️ Tech Stack

* HTML5

## ⚙️ How It Works

1. User selects **Skills**
2. User selects **Interests**
3. User selects **Education Level**
4. User selects **Work Style Preferences**
5. Click **Generate Career Roadmap**
6. AI matches profile with careers
7. Displays:

* Best Career Matches
* Match Percentage
* Why This Career Fits
* Required Skills
* Learning Courses
* Future Scope
* Career Roadmap

* Calculates match percentage
* Filters based on education
* Applies interest weighting
* Suggests roadmap accordingly
